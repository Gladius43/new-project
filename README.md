# Thrones Clash (MVP)

Локальний 1v1 прототип покрокової стратегії у стилі HoMM3 з фентезі-сетингом, натхненним інтригами великих домів.

## Що вже є

- Стартовий екран: вибір сторони і карти.
- Глобальна карта 10x7.
- 2 сценарії карти: `Королівський тракт` і `Крижаний рубіж`.
- Два гравці: людина проти AI.
- Шахти і міста, захоплення об'єктів, щоденні ресурси.
- Тактичний бій стеків (черга ходів за швидкістю).
- Екран міста:
  - будівництво споруд;
  - 1 споруда на місто за день;
  - найм юнітів залежить від побудованих споруд.
- SVG-іконки героїв/об'єктів і окремі SVG-спрайти юнітів.
- Збереження/завантаження через `localStorage`:
  - `Зберегти` / `Завантажити` в грі;
  - `Завантажити сейв` на стартовому екрані.

## Локальний запуск (браузер)

1. Відкрити `index.html` у браузері.
2. На стартовому екрані обрати сторону і карту.
3. Натиснути `Почати кампанію`.

## Команди проєкту

- Встановити залежності: `npm install`
- Smoke-тест: `npm run test:smoke`
- Запуск як desktop app (Electron): `npm start`
- Збірка DMG: `npm run build:dmg`

## Готовий DMG

Після збірки файл лежить тут:

- `dist/Thrones-Clash-0.1.0.dmg`

Примітка для macOS: застосунок без Developer ID-підпису, тому при першому запуску може знадобитись `Right Click -> Open`.

## Де міняти контент

- Логіка гри: `game.js`
- Стилі: `styles.css`
- Іконки карти/героїв: `assets/icons/*.svg`
- Спрайти юнітів: `assets/units/*.svg`
- Сценарії карт: `MAP_PRESETS` у `game.js`
- Будівлі міста: `TOWN_BUILDINGS` у `game.js`
- Стати юнітів і ціни: `unitCatalog` у `game.js`

## Тест

Є e2e smoke-тест у `tests/smoke.spec.js`:

- старт кампанії;
- збереження;
- завантаження сейву;
- відкриття екрана міста;
- будівництво/найм;
- перехід до наступного дня.

## Важливе про IP

Для публічного або комерційного релізу використовуй власні назви/персонажів/арт або ліцензований IP.
